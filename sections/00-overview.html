Prerequisites

This guide assumes you have a working familiarity with the following topics. If any of these feel unfamiliar, follow the linked resources before diving into the main content.

  • Basic Linux Administration Comfortable with the shell, systemd services, package management, user permissions, and reading logs. You should be able to SSH into a server and troubleshoot a failed service without a GUI.
  • Networking Fundamentals (TCP/IP, Subnetting) Understand the OSI model through Layer 4, IPv4 subnetting and CIDR notation, basic firewall rules, DNS resolution, and how a packet traverses a routed network.
  • Basic Understanding of Virtualization Know what a hypervisor does, the difference between Type-1 and Type-2 virtualization, and why hardware-assisted virtualization (VT-x / AMD-V) matters for performance.
  • Familiarity with Containers and Kubernetes Concepts Understand container images, container runtimes, pods, deployments, services, and namespaces. You do not need to be a Kubernetes expert—this guide will cover GPU-specific orchestration in depth.
